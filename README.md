# poapper-homepage

PoApper club homepage.

## Instructions

### Development

```shell
npm run develop
```

### Building

```shell
npm run build
```

## Technology Stack

### Frontend

- React with Gatsby
- CSS-in-JS via styled-components

### Tooling

- Formatted and linted automatically via lint-staged
- Formatted with prettier
- Linted with eslint and stylelint
